# translatorbot
Перевод текста
# Установка библиотек
Войдите в папку с ботом и введите
`pip install -r requirements.txt`
# Изменение токена
Зайдите в main.py и измените null на свой токен
# Запуск
ну тут слишком легко, 
`python main.py`
